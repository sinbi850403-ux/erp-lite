# Auth Regression QA Checklist

이 문서는 로그인 회귀 테스트를 고정된 체크리스트로 운영하기 위한 기준입니다.

## 자동 테스트

- `AUTH-001` 하드 리프레시(Ctrl+F5) 직후 첫 로그인 가드
- `AUTH-002` 비정상 세션 토큰(빈 토큰/누락 uid) 가드
- `AUTH-003` 유효한 세션 토큰 + uid bootstrap 허용

실행 명령:

```bash
npm run test:auth
```

## 수동 테스트

- `AUTH-004` 로그아웃 후 즉시 재로그인
- `AUTH-005` 느린 네트워크 환경에서 타임아웃/재시도 동작

수동 테스트는 배포 전 체크리스트로 반드시 확인합니다.
